About

This is the Material Design section. Each tab describes the elements used and where to find them on the site.

The Nav Toolbar is used at the top of the page, with the toggle button for the Side-Nav and Menu button for switching UI libraries.

The About section of the Side-Nav (here) contains the Tab layout element with Material Icons in one of the tab labels. I used an Indeterminate Progress Bar that goes away when the user is 'finished reading'.

The Form section contains DatePicker, Input, Radio, Checkbox, Select, Slide Toggle, and Button elements integrated with the ngForm module for validation and data-binding. A Determinate Progress Bar is used for indicating the completion of the form before the submit button is enabled. On submit a SnackBar element is used as an alert message, the form fields reset, and an http request is made through a service to post the data.

The Display section uses a service to make a get request and display the entries using the Card layout element with Chips for various fields. There is also a Dialog Modal used to insert credit card info for certain entries. An Indeterminate Progress Spinner is used that the user can stop by finishing reading.
This application is currently using MongoDB as the database, with the intention of switching to ReThinkDB. Node is used for the basic server setup, and express for the routes.
